## About This Project

The NFC Resident Profile System enables staff to tap a small NFC tag on a resident’s door frame with any smartphone, instantly opening a structured resident profile in the browser. Designed specifically for Bluecare Talleyhaven to support agency staff, casual staff, and visiting clinicians.
